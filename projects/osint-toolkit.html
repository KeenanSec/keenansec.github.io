NETSCANNER

VIEW ON GITHUB → PYTHON NETWORKING RECON

NetScanner is a lightweight Python-based network reconnaissance tool I built to speed up the initial enumeration phase in CTF environments. It wraps common scanning techniques into a single clean CLI interface.

// FEATURES

Fast TCP port scanning with configurable thread count, service version detection, basic banner grabbing, and structured JSON output for piping into other tools.

// USAGE

$ python3 netscanner.py -t 192.168.1.1 -p 1-1000 --threads 100
[*] Starting scan on 192.168.1.1
[+] 22/tcp   OPEN  OpenSSH 8.2
[+] 80/tcp   OPEN  Apache httpd 2.4.41
[+] 443/tcp  OPEN  Apache httpd 2.4.41
[*] Scan complete. 3 open ports found.

// WHY I BUILT IT

I wanted to understand exactly what was happening under the hood of tools like nmap. Building this from scratch using Python's socket library taught me more about TCP handshakes and service detection than reading any documentation.

// INSTALLATION

$ git clone https://github.com/YOUR_USERNAME/netscanner
$ cd netscanner
$ pip install -r requirements.txt
$ python3 netscanner.py --help
